# Smart Dine 
Suggests restaurant based on user's mood. (Use chatbot to get suggestions)

## Project Setup

This project uses **Vite**, **TypeScript**, and **Tailwind CSS**.

### Getting Started

#### 1. Clone the repository
    git clone <your-repo-url>
    cd <your-project-folder>

#### 2. Install dependencies
    npm install

#### 3. Start the development server
    npm run dev

### Environment Variables

Create a `.env` file in the project root:

    OPENROUTER_API_KEY=your_api_key_here

Make sure your API key never gets committed.

## Tech Stacks

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
